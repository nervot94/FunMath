# FunMath 
FunMath is een educatief rekenspel ontwikkeld voor leerlingen van groep 5-6 (10-12 jaar). 
Het spel helpt kinderen om op een leuke en uitdagende manier te oefenen met rekenen.

## Doel
Dit project is ontwikkeld voor **Basisschool de Regenboog** als onderdeel van een opdracht voor Software4you. 

## Functionaliteiten
- Optellen, aftrekken, vermenigvuldigen en deel sommen
- 3 levens, verlies een leven bij een fout antwoord
- Scores, verdien een punt wanneer je een goed antwoord heb
- Je ziet een groene flits als je antwoord goed is, zo niet dan zie je een rode flits.
- De applicatie kan makkelijk gebruikt woorden van leerlingen tussen de 10 en 12 jaar.

## Hoe te Spelen
1. Een willekeurige rekensom verschijnt op het scherm
2. Typ het antwoord in het invoerveld
3. Druk op "Check Antwoord" of druk op Enter
4. Bij een correct antwoord stijgt je score met 1 punt
5. Bij een fout antwoord verlies je 1 leven
6. Het spel eindigt wanneer je geen levens meer hebt en begint automatisch opnieuw met een nieuwe som.

## Installatie & Uitvoeren
1. Clone de repository:
  ```bash
  $ git clone https://github.com/nervot94/FunMath
  ```
2. Open het project in Visual Studio 2022 of hoger
3. Bouw het project (Build > Build Solution)
4. Start de applicatie (F5 of Run)
